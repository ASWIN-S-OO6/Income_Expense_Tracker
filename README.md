# Income Expense Tracker

Income Expense Tracker is an open-source Flutter application designed to help users manage monthly income and expenses efficiently.

## Features

- Separate profiles for:
    - Individual users
    - Company accounts

- Separate monthly books for organized financial tracking

- Add, edit, and delete income and expense entries

- View monthly summaries and financial records

- Data stored locally on the device (offline support)

- Import and export functionality for backup and data transfer

## Privacy

This app:
- Stores all data locally on the device
- Does not use analytics or tracking services
- Does not include advertisements
- Works completely offline

## Build Instructions
flutter clean

flutter pub get

flutter build apk --release
